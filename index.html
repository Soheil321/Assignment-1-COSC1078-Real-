Project Idea

Multi Host IP Looper

The idea behind this project is to create a program which will make your IP address virtually untraceable. Normal VPNs will just host your IP in a different location which although its safe, is still possible to trace through the company hosting. The MHIPL or Mipl, creates an untraceable IP by quite literally bouncing your IP address off of multiple site hosts at a constant rate so that it can never truly be pinned down anywhere. The Mipl is a program that is created to ensure internet safety and privacy at the highest level, ensuring your information is always protected. Our world is one where technology is ever-expanding with more and more of our information both professional and personal is up in the cloud supposedly secure. The reality is that with all your valuable information up in the cloud, the right people with the right skillset are going to be able to access that information and know everything about you and this is something that happens all the time. VPNs are the current commercial tool for data encryption and protection however even VPNs can be vulnerable and are not meant to run continuously and thus the Mipl will be the next step in cyber security. The MHIPL or the Mipl is the future of commercial cyber security. Utilizing multiple databases and DNS server hosts around the world, this program's function isn't to just mask your IP Address in one single location but to instead as the name spells out, Loop your already encrypted IP address through multiple server hosts at a constant rate so that there is not possibility for your IP to be tracked or pinned down in one place like a sitting duck. You wouldn't have to manually switch to another host for protection as you are automatically constantly being looped and bounced off of multiple hosts around the world. Using already existing public key encryption that all VPNs use, your IP address will be safe. The difference is that your IP will eb constantly moving around between server hosts so the potential hackers can't even trace it let alone try and gain access to it. It's a simple matter of just booting up your computer and switching on Mipl and just like that your IP address is moving around the world at a constantly fast rate. There is also no pattern to trace for hackers who may want to try predicting where your IP may end up and accessing it there as the locations your IP is moving through are decided at random via a random location generator. All of this at its base level will be coded through python using proprietary encryption software. The combination of multi-layered encryption and a constantly moving masked IP address makes you virtually untraceable and thus fully protected on the net. This program will both change the scape of network protection and cybersecurity and allow the general consumer to have the peace of mind knowing that they'll be safe browsing the net. The only weakness would the constant language changes that would occur when switching between servers so fast but there could be workarounds for this. To make this project work we would need multiple running server banks on home ground as well as access to multiple server hosts worldwide. That and using anaconda navigator, google collab, Jupiter notebook or any other programs to code python on as well as access to public key encryption software. Also the use of SQL software to create a database to organize the networks and manage the servers via a database Using these programs and servers is all we would need to get this project off the ground. Skills required would be an in-depth knowledge of database management and SQL software, in-depth knowledge of and experience with coding in python, in-depth knowledge of encryption software and using python to develop encryptions. Another important skill would be managing servers and assembly of servers to provide technical support should something go wrong with any of them. Overall this project will change the landscape of digital privacy and protection in a bid to keep our information safe and private from potential hackers. This will surpass VPNs as the go to consumer level digital privacy and protection as it would ensure total protection on the internet and the cloud.
